# RapidResumes 🚀

A fast, free resume builder — 13+ templates, live preview, PDF export.

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deploy to Vercel

Push to GitHub, then connect repo at vercel.com — zero config needed.

## Push to existing GitHub repo

```bash
# In the unzipped folder:
git init
git remote add origin https://github.com/ShiaEden/RapidResumes.git
git add .
git commit -m "fix: resolve all build errors, add logo, clean imports"
git push origin main --force
```

Then in Vercel dashboard → Deployments → Redeploy.
